## Reflektion
Jag lärde mig att container-säkerhet handlar om att minimera attackytor och använda minst privilegium. 
Att scanna bilder med Trivy hjälper att hitta kritiska sårbarheter. 
SBOM är viktigt för att se alla komponenter och snabbt kunna åtgärda säkerhetsproblem. 
Med Gatekeeper kan man säkerställa att Kubernetes-resurser följer policy redan vid skapande. 
Det förändrar arbetsflödet genom att integrera säkerhet och compliance tidigt. 
Utvecklare måste tänka på labels och regler innan deployment. 
Det ger en mer konsekvent och säker miljö och minskar risken för felkonfigurationer.
